ConfigureCapability DEVVAR_LONGSTRINGARRAY DEV_VOID OPERATOR Configures number of instances for each capability. If the capability exists,
it increments the configured instances by the number of instances requested,
otherwise an exception will be raised.
Note: The two lists arguments must be of equal length or an exception will be raised.
DeconfigureAllCapabilities DEV_STRING DEV_VOID OPERATOR Deconfigure all instances of the given Capability type. If the capability type does not exist an exception will be raised,
otherwise it sets the configured instances for that capability type to zero.
DeconfigureCapability DEVVAR_LONGSTRINGARRAY DEV_VOID OPERATOR Deconfigures a given number of instances for each capability. If the capability exists,
it decrements the configured instances by the number of instances requested,
otherwise an exceptioin will be raised.
Note: The two lists arguments must be of equal length or an exception will be raised

GetVersionInfo DEV_VOID DEVVAR_STRINGARRAY OPERATOR Array of version strings of all entities modelled by this device.
(One level down only)
Each string in the array lists the version info for one entity
managed by this device.
The first entry is version info for this TANGO Device itself.
The entities may be TANGO devices, or hardware LRUs or
anything else this devices manages/models.
The intention with this command is that it can provide more
detailed information than can be captured in the versionId
and buildState attributes, if necessary.
In the minimal case the GetVersionInfo will contain only the
versionId and buildState attributes of the next lower level
entities.
