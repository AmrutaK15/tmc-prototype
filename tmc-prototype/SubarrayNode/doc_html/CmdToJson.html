Command ToJson :
    Returns a JSON translation of this device.
    Defaults for empty string argin gives commands and metris:
    with_value:false,
    with_commands:true,
    with_metrics:true,
    with_attributes:false



ToJson Definition
Input Argument Tango::DEV_STRING Requests the JSON string representing this device, can be filtered
by with_commands, with_metrics, with_attributes and
with_value. Defaults for empty string argin are:
{`with_value`:`false`, `with_commands`:`true`,
`with_metrics`:`true, `with_attributes`:`false}
Output Argument Tango::DEV_STRING The JSON string representing this device,
filtered as per the input argument flags
